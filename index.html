Mi az a CSS és mire használjuk?

A CSS (Cascading Style Sheets) egy leírónyelv. A weboldalak stíluskészletét határozza meg tehát azt, hogy az egyes elemek hogyan nézzenek ki a weblapon.
A div az alapértelmezetten 100% széles és 0 px magas bekezdés szintű elem aminek nincsen margója.
Az ID egy attributum, mint a képeknél az src vagy a linkeknél a href
Style 3 féle helyen lehet inline, .css vagy itt bárhol egy style tagben

Szelektorok (az elemek kijelőlésre szolgálnak)

ID (azonosító)

Az ID egy egyedi azonosító. Egy tagnek csak egy id-ja lehet, egy id-t csak egyetlen tag kaphat meg.

Class (osztályba sorolás)

A class az osztályba sorolást jelenti. Egy tag-et lehet több osztályba sorolni is, és egy tag kaphat több osztályt is.
Ha több class-t szeretnénk megadni egy tag-nek, akkor az lehetséges de viszont
szóközzel elválasztva. pl.: class="square blue-bg" vagy class="square red-bg"
